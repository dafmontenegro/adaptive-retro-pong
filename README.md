# adaptive-retro-pong
El proyecto consiste en una nueva version del juego retro PONG lanzado por Atari en 1972. Esta nueva interpretacion del juego hace uso de p5.js (una biblioteca de JavaScript para la programaci√≥n creativa [1]), y tiene como propuesta, la implementacion de una dificultad adaptativa que busca siempre suponer un reto para el jugador, haciendo el juego mas facil o dificil de acuerdo a las habilidades del jugador durante el desarrollo de la partida.
